# Cara Inisialisasi Git untuk Project Portfolio

---

## 1. Pastikan Git Sudah Terinstall

Sebelum memulai, cek apakah Git sudah terpasang di komputer.

```bash
git --version
```

Jika Git sudah terinstal dengan benar, perintah ini akan menampilkan versi Git.

## 2. Buka Folder Project Portfolio

Masuk ke folder tempat project portfolio kamu berada.

Contoh (Windows / CMD / PowerShell):

```bash
cd C:\Users\NamaKamu\Documents\portfolio
```

Contoh (MacOS / Linux terminal):

```bash
cd ~/Documents/portfolio
```

> Perintah `cd` digunakan untuk berpindah direktori ke folder project.
